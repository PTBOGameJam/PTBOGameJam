Business Event Information

The PTBO Game Jam is a bi-annual game jam event in the city of Peterborough, Ontario, Canada.

Run by industry professionals with the support of the local tech industry, the PTBO Game Jam is a not-for-profit event that strives to build a game development industry locally while engaging the local community with the games industry to form long term relationships.

The event was created by dotBunny with the intent that it grow to a point where it can stand on its own, fully supported by the community. A press kit is available for download here, with a brief also available separately.
